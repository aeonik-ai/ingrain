# Philosophy

Most agent memory products are framed as recall. Ingrain is framed as learned experience.

Recall answers: what did I see before?

Learned experience answers: what should change next time?

That distinction matters because autonomous agents are judged by behavior, not by whether they can search a transcript.

The public claim is intentionally small:

> Give your agents learned experience.

Ingrain does not claim model-weight learning, consciousness, or autonomy. It compiles local execution history into behavior-shaping context that can be audited and evaluated.

It also does not claim active intent. Hermes owns goals, subgoals, missions, Kanban, scheduling, and task lifecycle. Ingrain owns the experience that should inform how that intent gets executed.
