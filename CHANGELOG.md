# Changelog

All notable changes are kept here. For the full per-commit history, `git log`.

## v0.2.0 ‚Äî 2026-05-20

**Architectural shift: deterministic regex compiler ‚Üí LLM consolidator.** Driven by external benchmark evidence; full arc in [`docs/research-arc.md`](docs/research-arc.md).

### Added

- **`integrations/hermes_consolidator/`** ‚Äî LLM-driven event-to-card consolidator that shells out to `hermes -z`. No API keys; uses whatever model Hermes is configured against.
- **`ingrain consolidate`** CLI command ‚Äî runs the consolidator on recent events.
- **`ingrain why <query>`** CLI command ‚Äî audit trail showing source events for any card matching the query. Differentiator over MemGPT/Mem0/Letta/Zep.
- **`integrations/hermes_plugin/`** ‚Äî Hermes lifecycle plugin: records tool calls on `post_tool_call`, runs `ingrain consolidate` on `on_session_end`. `ingrain install hermes-plugin` to enable.
- **`ingrain record --batch <jsonl>`** ‚Äî bulk-ingest path (~400√ó faster than per-event subprocess for long traces).
- **`integrations/sandbox_universe/lane.py:IngrainLLMSidecarLane`** ‚Äî new `ingrain-llm-sidecar` entry point.
- **`docs/research-arc.md`** ‚Äî 2000-word narrative writeup of the engineering arc.
- **`AUDIT.md`** ‚Äî living public-readiness checklist.
- **`CONTRIBUTING.md`** ‚Äî what kinds of contributions are wanted.
- **`assets/demo.gif`** + `assets/demo.tape` ‚Äî CLI walkthrough animation (generated by `vhs`).
- **Tests**: 20 new (consolidator, plugin, `ingrain why`, batch record). 63 total, all stdlib `unittest`.

### Changed

- **Sidecar lane now emits Hermes default memory content** (was previously emitting only a character count). The sidecar is now empirically `‚â• default` by construction ‚Äî verified on the n=50 LongMemEval Oracle run with 12 wins / 0 losses / 38 ties.
- **CLI `ingrain practice --no-compile`** ‚Äî skip the deterministic compile step when the consolidator already wrote cards.
- **`docs/` reorganized** ‚Äî pre-v0.2 docs moved to `docs/archive/`; canonical docs stay at the top level.
- **Repo URLs** in `pyproject.toml` kept pointing at `aeonik-ai/ingrain` (the company-owned canonical home). An intermediate revision pointed at a personal mirror; that's been reverted now that the Aeonik AI org is the canonical home.
- **README rewritten** ‚Äî leads with the ELI10 ("memory vs learned experience") framing, the n=50 LongMemEval evidence, and the animated demo. The `Memory vs. Learned Experience` distinction is the framing the rest of the repo follows.

### Removed

- **`integrations/hermes/`** top-level dir ‚Äî a 5-line shim + duplicate README that was confusing readers about where the canonical provider code lives.
- **`WORKLOG.md`** ‚Äî preserved at `~/.aeonik-ingrain-WORKLOG.md` locally for reference; git history retains it for auditability.

### Fixed

- **Consolidator subprocess env propagation** ‚Äî the inner `hermes -z` call now strips the outer lane's sandboxed `HERMES_HOME` so it can find user auth at the real `~/.hermes`.
- **`les_hard.py` test fixture path** parameterized as `$HOME/.hindsight` (was `/Users/benlloyd/.hindsight`).
- **All committed evidence paths redacted** to placeholder strings (`<home>`, `<hermes-root>`, etc.).
